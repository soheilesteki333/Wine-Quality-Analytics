# Wine-Quality-Analytics
# End-to-End Wine Quality Analytics

Predictive modeling for wine quality using classification & regression with Python & ML.  
Includes Exploratory Data Analysis (EDA), feature engineering, and Explainable AI insights.

## Features
- Data cleaning & preprocessing
- EDA
- Classification & regression modeling
- Model evaluation (Accuracy, RMSE)
- Explainable AI (feature importance visualizations)

## Tech Stack
- Python (pandas, scikit-learn, matplotlib, seaborn)
- Jupyter Notebook
